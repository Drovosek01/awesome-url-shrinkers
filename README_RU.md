## Что это за репозиторий

Этот репозиторий - это список сервисов для укорачивания длинной ссылки в короткую путем перенаправления перехода с короткой ссылки на длинную. В общем сервисы, которые перенаправляют переходы по ссылкам.

Подробнее о концепции "Awesome" списков можно почитать [здесь](https://github.com/sindresorhus/awesome/blob/master/awesome.md)
